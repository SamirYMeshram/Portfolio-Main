Multipage Portfolio Expansion
============================

Files included:
- index.html / styles.css / script.js : original landing page updated so nav links open the new dedicated pages
- about.html / about.css / about.js : deep about page
- journey.html / journey.css / journey.js : detailed timeline and growth page
- skills.html / skills.css / skills.js : deep skill architecture page with tabs and radar chart
- projects.html / projects.css / projects.js : filterable case studies with detail drawer and testimonials
- services.html / services.css / services.js : service system, estimator, packages, FAQ
- contact.html / contact.css / contact.js : detailed contact page with quick brief builder and structured mailto form

How to use:
1. Keep all files together in the same folder.
2. Open index.html in a browser.
3. Navigate through the pages from the top navigation.

Notes:
- The dedicated pages keep the same visual universe as the home page while adding page-specific interactions.
- Contact uses mailto for front-end-only functionality. Connect a backend or form service for production form submission.
